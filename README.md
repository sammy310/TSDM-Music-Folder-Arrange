# TSDM-Music-Folder-Arrange
TSDM 노래 폴더, 파일들을 정리해주는 프로그램입니다

### 사용법
1. TSDM 폴더들이 들어있는 폴더를 지정
2. 정리 버튼을 클릭

### How To Use
1. Specify the folder where containing TSDM folders
2. Click the 정리(Arrange) button
